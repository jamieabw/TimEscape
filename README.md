# No time to explain.. Just Jump!

A fast paced, challenging platformer with infinite, randomly generated levels and brutal time pressure. Race to the exit portal, gain coins to buy abilities, and defeat enemies to earn the extra seconds you need to survive.

---

## Requirements

- Python 3.11 recommended

---

## Installation

From the root folder of the project:

```
pip install -r requirements.txt
```

---

## Running the game

### Windows

```
cd src
python main.py
```

### Mac / Linux

```
cd src
python main.py
```

---

## How to play

- Find the exit portal to advance to the next level before you run out of time.
- Kill ghosts to gain extra time.
- Avoid deadly traps and enemies.
- Upgrade your abilities by gaining coins
- Survive as long as possible and reach the highest level you can.
  
---

## Controls

- **A / D** - Move left and right
- **Space** - Jump
- **Left click** - Attack
